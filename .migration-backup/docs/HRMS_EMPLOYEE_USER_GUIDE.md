# HRMS Employee User Guide

Date: 2026-06-26
Audience: Employees using self-service module

## 1) Where to start

Open:
- `/hrms/v2/self-service`

You can also access direct tabs:
- `/hrms/v2/self-service/profile`
- `/hrms/v2/self-service/attendance`
- `/hrms/v2/self-service/leave`
- `/hrms/v2/self-service/calendar`
- `/hrms/v2/self-service/payroll`
- `/hrms/v2/self-service/work-mode`

## 2) Self-service navigation

### My Profile
- Review and update personal profile information as allowed.

### My Attendance
- Review attendance timeline.
- Submit correction requests when needed.

### My Leave
- Apply leave requests.
- Track leave approval status.
- Review leave balance.

### My Calendar
- View leave and holiday planning context.

### My Payroll
- View payroll and payslip-related information.

### Work Mode
- Manage office/remote/work-from-home preference where enabled.

## 3) Common workflows

### Apply leave
1. Go to My Leave.
2. Enter leave dates and type.
3. Submit request.
4. Track status in the same module.

### Attendance correction
1. Go to My Attendance.
2. Create correction request with reason.
3. Wait for HR review outcome.

### Payroll check
1. Go to My Payroll.
2. Verify current cycle details.
3. Download available payroll documents if enabled.

## 4) Access behavior

- Employee role is restricted to own data only.
- HR admin modules are hidden for employee role.
- If access is denied unexpectedly, contact HR Admin.

## 5) Troubleshooting

- If page is blank or data is missing, refresh once and re-login.
- If self-service tab errors continue, report timestamp and tab route to support.
- For payroll discrepancies, raise an HR ticket with month/year and screenshot.
